# ATmega2560 Embedded Driver Library

## Overview

This repository contains reusable embedded driver libraries and example applications developed for the ATmega2560 microcontroller using register-level (bare-metal) programming.

The objective of this repository is to understand AVR peripheral architecture through direct register manipulation without relying on Arduino built-in functions.

Currently, the repository includes:

* GPIO Driver Library
* Timer Driver Library
* Register-Level Example Applications

These projects demonstrate polling, interrupt-driven programming, PWM generation, timer configuration, peripheral interfacing, and embedded firmware development.
